# Traits Genealogy

## Running

```
$ python3.6 main.py
```

Paramaters are found in ```main.py```, and are described there. ```fitnessfunctions.py``` contains hard-coded functions to choose from. Feel free to add your own!

## Dependencies

- numpy
- matplotlib
- python3.6
